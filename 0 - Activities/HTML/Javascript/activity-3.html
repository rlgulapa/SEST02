ACTIVITY 3: Guess the Secret Number

  1. Open your JavaScript development environment or console.
  2. Declare a variable named secretNumber and assign it a random integer between 1 and 10.
  3. Declare a variable named attempts and set it to 0. This variable will keep track of the number of attempts made by the player.
  4. Declare a variable named guessedNumber to store the player's guesses.
  5. Display a welcome message to the player using console.log().
  6. Start a do-while loop to repeatedly ask the player for their guess until they guess the correct number.
  7. Inside the loop, prompt the player to enter their guess using prompt() function and store it in the guessedNumber variable.
  8. Use an if statement to check if the guessedNumber is lower than the secretNumber.
    1. If it is lower, display the message "Too low! Try again." using console.log().
  9. Use an else if statement to check if the guessedNumber is higher than the secretNumber.
    1. If it is higher, display the message "Too high! Try again." using console.log().
  10. Use an else statement to handle the case when the guessedNumber matches the secretNumber.
    1. Display the message "Congratulations! You guessed the correct number: [secretNumber]" using console.log().
    2. Display the message "It took you [attempts] attempts." to let the player know how many attempts they took.
  11. Increment the attempts variable by 1.
  12. Repeat steps 7-11 until the guessedNumber matches the secretNumber.
  13. End the game.
